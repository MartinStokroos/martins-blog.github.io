Integrating the CMSIS (DSP) library in STM32CubeIDE

Published Mar 12, 2021

I couldn’t find an option to download or integrate the ARM CMSIS library in STM32CubeIDE with a single mouse click. While working with the STM32F4 family controllers, I would like to make use of the DSP functions with the on chip FPU. The informative Youtube channel YetAnotherElectronicsChannel explains how to manually add the CMSIS library into the project. I found a way to make the library selectable from the CubeMX configurator tool built-in the IDE.
